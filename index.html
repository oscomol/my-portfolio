Recent Projects

Clinic Online Appoitment System

This project involves the development of an Online Clinic Reservation System. The system allows patients to book appointments with their preferred healthcare providers through a user-friendly interface, manage their schedules, and receive reminders. It is designed to streamline the process for both patients and clinic staff, ensuring efficient time management and enhancing the overall patient experience. The project was developed using Laravel, MySQL, jQuery, and AdminLTE. It is fully functional and mobile responsive. Developing the system enhanced my skills in Laravel and jQuery. Key features of the system include email authentication and notifications, automated email updates through Laravel's built-in task scheduling, and a simple reservation process that minimizes booking errors while maintaining clinic efficiency. Additionally, the system provides an intuitive dashboard for clinic staff to effectively manage appointments and patient records.

Judging Contest Automation

This project involves the development of a system for automating judging contests using Laravel. The system is designed to streamline the entire judging process, from registration and submission to scoring and results. By leveraging Laravel's robust framework, the system ensures a user-friendly interface for contestants and judges alike, efficient handling of large volumes of data, and secure management of contest information. Key features include automated score calculation, real-time progress tracking, and comprehensive reporting capabilities, all aimed at enhancing the accuracy, transparency, and efficiency of contest judging.

Capstone project

IoT based ph level monitoring and adjustment system

Developing an intelligent and automated pH monitoring and adjustment system offers a crucial solution to the challenges faced in plant tissue analysis. This transformative system improves laboratory efficiency and accuracy by automating the pH monitoring and adjustment process. The project was built using Arduino on the machine side, Node.js on the server side, and React Native for the mobile application. As the developer of the group during our capstone project, I honed my skills in JavaScript, particularly with React Native and Node.js, as well as programming with Arduino. Key features of the system include real-time updates, push notifications, automatic pH level adjustments, and comprehensive pH level record management.

E-Commerce Web Application

Developed a robust e-commerce web application designed to deliver a seamless and engaging online shopping experience. The system features a user-friendly interface, comprehensive product management, and efficient product browsing capabilities. It incorporates a real-time messaging system to facilitate immediate communication between users and support teams.
